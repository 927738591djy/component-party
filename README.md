> 通过语法和特性快速概述Web组件JS框架

**网站: <https://component-party.lainbo.com/>**

## 🤔 Why ?

许多JS开发者并未对所有现有的JS框架及其自身的语法和特性有一个良好的概览。我们如何解决这个问题呢？开发者们喜欢通过例子来了解框架的概览。这是在深入学习之前的快速介绍。
